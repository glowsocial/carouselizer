# Carouselizer.ai

Free carousel generator for social media — a lead magnet micro-site by [Glow Social](https://glowsocial.com).

## What is this?

Carousels get 3× more engagement than static posts. This site lets business owners request a free custom carousel by entering their business info and a topic. The carousel is delivered by email.

## How it works

1. Visitor describes their business and picks a topic
2. We create a tailored multi-slide carousel
3. They get it by email, ready to post on LinkedIn, Instagram, or Facebook

## Tech stack

- Static HTML, CSS, and vanilla JavaScript
- Hosted on GitHub Pages
- Custom domain: [carouselizer.ai](https://carouselizer.ai)
- Form submissions handled via Paperform → ConvertKit
